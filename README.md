# React-on-Express

> This is an demo of web application

## dev

```
$ git clone https://github.com/davidkoojohn/react-on-express.git
```

## Install npm packages

```
$ yarn
```
Or
```
$ npm install
```

## Start dev Server

```
$ yarn start
```
Or
```
$ npm run start
```

Then `open http://localhost:7777`
